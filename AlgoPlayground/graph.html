Graph Traversal Algorithms

Breadth-First Search (BFS)

BFS explores the graph level by level, visiting all neighbors of a node before moving to the next level. It's implemented using a queue and is particularly useful for finding the shortest path in unweighted graphs.

Depth-First Search (DFS)

DFS explores as far as possible along each branch before backtracking. It can be implemented recursively or using a stack. DFS is useful for topological sorting, cycle detection, and solving puzzles.
