BFS Pseudocode

function BFS(start):
  queue = [start]
  visited = set(start)
  while queue not empty:
    node = queue.shift()
    for neighbor in neighbors(node):
      if neighbor not in visited:
        visited.add(neighbor)
        queue.push(neighbor)
        // mark neighbor as discovered
